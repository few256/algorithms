# 최대공약수와 최소공배수

\\( x, y \in \mathbb{Z} \\) 에 대해 아래의 \\( \operatorname{gcd}(x, y) \\)
는 \\( x, y \\) 의 최대공약수를 계산합니다.

\\( x \\) 와 \\( y \\) 는 서로의 최대공약수 \\( G \\) 에 대해
\\( x = x' G,\ y = y' G,\ (x', y') = 1 \\) 로 표현할 수 있고 이 때 \\( x, y \\) 의 최소공배수
\\( L \\) 은 \\( L = x' y' G = \frac{xy}{G} \\) 가 됩니다.
